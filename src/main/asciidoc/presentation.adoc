== Adding an OpenAPI Macro to a Confluence Page footnote:ref[Source: https://oharagroup.bitbucket.io/confluence-open-api/]
Example:
https://petstore.swagger.io/v2/swagger.json

. Start by creating/editing a page in Confluence
. Insert the macro either
* via the Macro Browser (search for "Open API" or "Swagger")
* or by simply typing either *{open-api* or *{swagger*
. Specify your Open API-compliant JSON or YAML definition either
* by typing or pasting JSON or YAML directly into the macro body
* in the macro configuration dialog, enter the URL of a publicly accessible JSON or YAML definition
